## 개요
- TG Wing Algorithm에서 푼 알고리즘 문제 보관 폴더
```
TG Wing Algorithm
    │   └── * root directory
    ├── ddolsoon
    │   └── 민성 알고리즘 폴더
    |        └── week 0 ...
    ├── jinuman
    │   └── 진우 알고리즘 폴더
    |        └── week 0 ...
    ├── sheep1500
    │   └── 석윤 알고리즘 폴더
    |        └── week 0 ...
    ├── minq
    │   └── 민규 알고리즘 폴더
    |        └── week 0 ...
    ├── etc
	└── * 기타 자료

```
* Algorithm 파일 업로드 제목 형식: year/month/date/-간단한제목
	* ex) 2017/08/05-알고리즘 N주차 폴더 생성 후,이 폴더 안에 알고리즘 스터디 자료를 업로드 한다.

## 스터디 진행방식
1. TG Wing 알고리즘에서 풀었던 알고리즘 문제 각자 week별로 폴더에 올려주기!
